## 6. 新增测试时要关注什么

新增读测试前，先确认：

- 地址是否正确
- ABI 是否是最新版本
- 方法是否是 `view` / `pure`
- 参数是否已经确认

新增写测试前，先确认：

- 方法名和参数顺序完全正确
- 你理解这会发起真实交易
- 是否可以先用 `staticCall` 做预检查
- 是否应该继续保持默认跳过状态
